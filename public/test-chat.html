üêõ Troubleshooting

Widget doesn't appear?

  • Press F12 to open browser console
  • Check for JavaScript errors
  • Verify server is running: http://localhost:3000/widget.js
  • Make sure you updated the webchat_id

Can't send messages?

  • Check browser console (F12) for errors
  • Verify webchat_id is correct
  • Check server logs in terminal

No AI response?

  • Check server logs for OpenAI errors
  • Verify OpenAI API key is valid
  • Wait up to 10 seconds (first response can be slow)
