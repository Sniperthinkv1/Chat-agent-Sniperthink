âš¡ Quick Start

Base URL: https://api.example.com (Production) | http://localhost:3000 (Development)

1. Add Phone Number

curl -X POST http://localhost:3000/users/YOUR_USER_ID/phone_numbers \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "id": "phone_123",
    "platform": "whatsapp",
    "meta_phone_number_id": "836990829491415",
    "access_token": "EAAxxxx...",
    "display_name": "+1 (234) 567-8900"
  }'

2. Create AI Agent

curl -X POST http://localhost:3000/users/YOUR_USER_ID/agents \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_id": "agent_123",
    "phone_number_id": "phone_123",
    "prompt_id": "asst_abc123",
    "name": "Customer Support Bot"
  }'

3. Add Credits

curl -X POST http://localhost:3000/users/YOUR_USER_ID/credits/add \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"amount": 1000}'
